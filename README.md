# Gu Lab website

Source for [gu-lab.net](https://gu-lab.net) — the Gu Lab at City of Hope.

Static HTML/CSS site, deployed via GitHub Pages from the `main` branch.

## Structure

- `index.html` — landing page
- `research.html` — research areas
- `members.html` — lab members
- `publications.html` — publications
- `join_us.html` — open positions
- `blog.html` — blog
- `css/` — stylesheets
- `fonts/` — web fonts
- `images/` — site images
- `CNAME` — custom domain configuration for GitHub Pages

## Editing

Edit the `.html` files directly. Push to `main`; the site rebuilds automatically.
